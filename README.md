# LeiloesTDSat

Descrição do Projeto

Este projeto é um sistema de gerenciamento de leilões desenvolvido como atividade prática para a Unidade Curricular 11 (UC11). O objetivo principal é aplicar conceitos de persistência de dados e versionamento de código, permitindo o cadastro, listagem e controle de produtos destinados a leilão.
Tecnologias Utilizadas

    Linguagem: Java

    Banco de Dados: MySQL

    IDE: NetBeans

    Versionamento: Git e GitHub
